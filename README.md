# Deal Flow Outbound System Cost Overview

This repository contains a single-page HTML cost overview for a deal flow outbound system proposal.

The page explains the strategy, scope, assumptions, and pricing for building a repeatable outbound pipeline across the Netherlands, UK, and US, focused on technology and agriculture investors, sellers, founders, and companies.

It includes interactive pricing controls for:

- Cold email infrastructure
- Lead sourcing methods
- Email finding, verification, and enrichment
- Contact volume
- Cold email marketing operations
- Signals OS
- One-time build costs
- Monthly recurring cost and ROI reference

Open `index.html` directly in a browser to view the proposal.
